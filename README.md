## 1. Diabetes Risk Prediction (Machine Learning)
An end-to-end Machine Learning pipeline designed to predict the likelihood of diabetes in patients based on medical metrics.

### Key Features:
* **Data Cleaning:** Replaced invalid zero values (Glucose, BMI, etc.) with statistical **Medians**.
* **Algorithm:** Implemented a **Random Forest Classifier**.
* **Preprocessing:** Used `StandardScaler` for feature normalization.
* **Model Persistence:** Saved the trained model using `joblib` for real-time predictions.

### Files:
* `Diabetes_Project.ipynb`: The main Python notebook.
* `diabetes_model.pkl`: The saved "brain" of the model.
* `diabetes_scaler.pkl`: The saved scaler for processing new data.
